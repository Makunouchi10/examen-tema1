# Assets

Esta carpeta contiene todos los recursos estáticos utilizados en el proyecto Canvas 2D Geometric Drawing.

## Estructura de la carpeta

```
assets/
├── css/      # Hojas de estilo
├── img/      # Imágenes
└── js/       # Archivos JavaScript
```

## Contenido

### css/
- **styles.css**: Estilos personalizados para el proyecto.
  - Color de fondo del cuerpo (#f4f4f4)
  - Estilo del borde del lienzo (3 píxeles de negro sólido con fondo blanco)

### img/
- **carro.png**: Favicon para el sitio web
- **image.png**: Imagen de referencia para el dibujo geométrico

### js/
- **main.js**: Código JavaScript para la representación 2D de Canvas
  - Contiene la lógica del dibujo geométrico
  - Dibuja formas mediante programación en el elemento canvas

## Uso

Estos recursos son referenciados por el archivo principal `index.html`:
- El CSS está vinculado en la sección `<head>`
- El JavaScript se carga al final del `<body>`
- Las imágenes se muestran en el contenido de la página
